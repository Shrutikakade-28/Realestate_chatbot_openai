# Mini Real Estate Analysis Chatbot - Full Project

This zip contains a Django backend and a React frontend.

Folders:
- backend/: Django project with API at /api/analyze/
- frontend/: React app (simple chat UI)

Quick start:
1. Backend:
   - cd backend
   - python -m venv venv; source venv/bin/activate
   - pip install -r requirements.txt
   - python manage.py migrate
   - python manage.py runserver

2. Frontend:
   - cd frontend
   - npm install
   - npm start

Sample Excel is included at backend/sample_data/real_estate_sample.xlsx
